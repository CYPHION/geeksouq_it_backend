# listed here — see requirements-postgres.txt. psycopg2-binary only ships

# to compile it from source (no matching wheel, e.g. a brand-new Python

# install -r requirements.txt` command — nothing in this file gets
# installed either, not just the Postgres driver. Keeping it out of the
# core file means Flask/SQLAlchemy/etc. always install cleanly regardless.
